## 1. 业务目标

实现患者管理功能，为医疗系统提供完整的患者信息生命周期管理能力，支持医护人员高效地新增、查询、修改、删除及查看患者详情，提升医疗机构的患者信息管理效率与准确性。

## 2. 用户角色

| 角色 | 说明 |
|------|------|
| 医护人员 | 负责日常患者信息的录入、查询和维护 |

## 5. 页面流程

```
患者列表页
  ├── [新增] → 患者信息录入表单 → 保存 → 返回列表页
  ├── [搜索/筛选] → 刷新列表
  ├── [点击行/详情] → 患者详情页
  │     ├── [编辑] → 患者信息编辑表单 → 保存 → 返回详情页
  │     └── [删除] → 二次确认弹窗 → 逻辑删除 → 返回列表页
  └── [导出] → 下载 Excel/CSV
```

## 7. 验收标准汇总

| 患者信息新增 | 成功录入一条患者记录，系统自动生成 Patient ID；身份证重复时给出明确错误提示 |
