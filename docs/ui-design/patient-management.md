# 患者管理模块 — UI/UX 设计文档

## 1. 信息架构

```
患者管理
├── 患者列表页（/patients）
│   ├── 搜索/筛选栏
│   ├── 患者数据表格（含操作列）
│   └── 分页控件
├── 新增患者页（/patients/new）
├── 患者详情页（/patients/:id）
└── 编辑患者页（/patients/:id/edit）
```

## 2. 患者列表页

### 2.1 布局

```
┌─────────────────────────────────────────────────────────────┐
│  患者管理                              [+ 新增患者]          │
├─────────────────────────────────────────────────────────────┤
│  姓名 [___________]  性别 [全部▼]  电话 [___________] [搜索] │
├────┬──────┬────┬────────────┬──────┬──────────────────────┤
│ #  │ 姓名 │性别│  出生日期  │ 电话 │       操作           │
├────┼──────┼────┼────────────┼──────┼──────────────────────┤
│    │      │    │            │      │ [查看] [编辑] [删除] │
│ …  │  …   │ … │     …      │  …   │         …            │
├────┴──────┴────┴────────────┴──────┴──────────────────────┤
│  共 N 条   « 上一页   第 P / T 页   下一页 »               │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 交互说明

- **搜索**：输入框失焦或按 Enter 触发查询，恢复至第 1 页。
- **删除**：点击「删除」弹出确认对话框，确认后执行软删除并刷新列表。
- **空状态**：无数据时显示「暂无患者信息，点击右上角新增」。
- **加载中**：表格区域显示 Spinner。

## 3. 新增 / 编辑患者表单

```
┌─────────────────────────────────────────┐
│  新增患者（或 编辑患者）                 │
├─────────────────────────────────────────┤
│  姓名 *   [_____________________]       │
│  性别 *   ○ 男  ○ 女  ○ 其他           │
│  出生日期 * [____-__-__] (日期选择器)   │
│  联系电话 * [_____________________]     │
│  身份证号   [_____________________]     │
│  地址       [_____________________]     │
│  过敏史     [_____________________]     │
│  备注       [____________________]      │
│                                         │
│           [取消]   [保存]               │
└─────────────────────────────────────────┘
```

- 必填字段标注 `*`，未填时按钮旁显示内联错误提示。
- 「保存」按钮在提交期间变为禁用+Loading 状态，防止重复提交。

## 6. 颜色与字体规范

| 元素           | 颜色            |
|----------------|-----------------|
| 主操作按钮     | `#1677ff`       |
| 危险操作按钮   | `#ff4d4f`       |
| 背景           | `#f5f5f5`       |
| 卡片背景       | `#ffffff`       |
| 主文字         | `#333333`       |
| 辅助文字       | `#666666`       |

- 字体：`-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- 基础字号：14px；标题：18px Bold。

## 7. 响应式要求

本期仅支持桌面端（≥ 1280 px 宽），移动端适配纳入后续迭代。
