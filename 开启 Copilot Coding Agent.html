Features

  • Editor preview features
    Enabled

    You can use preview features in your editor. Learn more.

  • Copilot Chat in GitHub.com
    Enabled

    You can use Copilot Chat in GitHub.com and Copilot for pull requests. Preview features are only available for paid licenses. Learn more about Copilot in GitHub.com.

  • Copilot CLI
    Enabled

    You can use GitHub Copilot for assistance in terminal.

  • Copilot in GitHub Desktop
    Enabled

    You can use GitHub Copilot for assistance in GitHub Desktop.

  • Copilot Chat in the IDE
    Enabled

    You can use GitHub Copilot Chat in the IDE to explain a piece of code, propose bug fixes, or generate unit tests in a chat interface from your editor.

  • Copilot Chat in GitHub Mobile
    Enabled

    You can use Copilot Chat in GitHub Mobile personalized to a codebase.

  • Copilot can search the web

    Copilot can answer questions about new trends and give improved answers, via Bing. See Microsoft Privacy Statement

  • Copilot can search the web using model native search Preview

    Copilot can answer questions using a model's built-in search capabilities.

  • Dashboard Entry Point

    Allows instant chatting when landing on GitHub.com

  • Copilot code review
    Enabled

    Use Copilot to review your code and generate pull request summaries.

    Automatic Copilot code review

    Use Copilot to automatically review all the pull requests you've created.

  • Copilot cloud agent
    Enabled

    Delegate tasks to Copilot cloud agent in repositories where it is enabled Learn more.

  • Copilot Memory Preview

    You can use Copilot Memory to remember repository context across Copilot agent interactions. This preview is governed by GitHub's pre-release terms.

  • MCP servers in Copilot
    Enabled

    Connect MCP servers to Copilot in all Copilot editors and Coding Agent.

  • Copilot-generated commit messages

    Allow Copilot to suggest commit messages when you make changes on GitHub.com. Learn more about Copilot-generated commit messages.

  • Copilot Spaces
    Enabled

    If enabled, you can view and create Copilot Spaces. When disabled, you cannot view or create any Copilot Spaces.

  • Copilot Spaces Individual Access
    Enabled

    If enabled, you can create individually owned Copilot Spaces. When disabled, you cannot create individual spaces.

  • Copilot Spaces Individual Sharing
    Enabled

    If enabled, you can share individually owned Copilot Spaces. When disabled, you cannot share individual spaces.
